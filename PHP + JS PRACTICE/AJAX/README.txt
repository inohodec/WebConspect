программа выводит записи из БД при помощи AJAX при нажатии на кнопку "Show meals", с возможностью их удаления из DOM или БД 
При нажатии на кнопку удаления из DOM? JS удаляет конкретную запись со сртаницы, соответственно не перезагружая её.
При нажатии на удаления из БД выполняеться AJAX запрос на сервер где php удаляет запись из БД и формирует ответ об удалении записи, а так же возвращает все оставшиеся.


ВАЖНО в папке PURE находится код программы без комментариев
