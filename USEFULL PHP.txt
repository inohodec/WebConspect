2 Список файлов/директорий по маске можно получит функцией glob('/img/*.png') : array

3 Для проверки существования в базе значения(например для исключения существования дубля имени и при его существовании добавление к непу к примеру _new) придется использовать полный цикл выполнения-получения запроса-данных 
	$conn->exec("SELECT id FROM table WHERE name='max'");    вернет 0 если значения нет, но из-за запроса SELECT mysql подготовит данные на вывод(типа данные не найдены), и дальнейшее выполнение запросов будет невозможно, т.к. может существовать только 1 активный запрос и пока он не будет получен дальнейшие запросы будут вываливатся в ошибку, в связи с тем что exec() возвращает только кол-во затронутых строк, зависший результат SELECT не даст нам работать.

	Выход - полный цикл!
	$result = $conn->query("SELECT id FROM table WHERE name='someone'");
	$id = $result->fetchAll(PDO::FETCH_ASSOC);
	if (empty($id)) {
		//do something
	}

так же при подготовленных запросах невозможно проверить результат с помощью:
	$stmt = $conn->prepare($query);
	$stmt->execute([$value]); 			//если сам синтаксис SQL запроса правильный, то вернет true, даже 									  
                                        //если к примеру SELECT ничего не нашел
